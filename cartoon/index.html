Модульный конструктор популярных мультсериалов для NETFLIX

АКТ I — ЗАВЯЗКА

Сцена 1: Улица / Бар "НеМозг" / Вечер
Сцена 2: Комната Персонажа №1 / Ночь
Раскадровка: 6 кадров
Статика → крупный план глаз → интерфейс → нажатие → затемнение → звуковой сдвиг.

Слов: ~90
Диалогов: 3 (ирония, экспозиция, рекламный крючок)
Задача: Представить героя, заложить сомнение в своей идентичности, зацепить абсурдом.
Техника: Бленд хроматических эффектов + дроновая озвучка, первая интеграция лейтмотива.

АКТ II — РАЗВИТИЕ

Сцена 3: Виртуальный Центр Подсознательной Инсталляции / День
Сцена 4: Интерфейсная загрузка личности
Раскадровка: 7 кадров
Персонаж №1 → загрузка → сцены идеального будущего → цветовая деформация → звуковая атака → белый шум → зеркало.

Слов: ~140
Диалогов: 5 (вопрос-ответ, философская подмена, завершающий триггер)
Задача: Продемонстрировать механизмы сервиса, ввести «вирус совершенства».
Техника: Использование синестезии (цвет/звук/речь), гипнотические петли, переход в кислотно-гладкий ритм.

АКТ III — КУЛЬМИНАЦИЯ

Сцена 5: Коворкинг / Утро
Сцена 6: Дом / Вечер
Раскадровка: 8 кадров
Вход → пустота в диалогах → отчуждение → одиночество → интерфейсы → взгляд → гаснет.

Слов: ~130
Диалогов: 3 (неловкий контакт, отказ, технический комментарий ИИ)
Задача: Эмоциональное обнажение. Демонстрация «смерти личности».
Техника: Уменьшение цветового шума, гашение динамики, аудио-паузирование лейтмотива.

АКТ IV — РАЗВЯЗКА

Сцена 7: Центр SubCogService / День
Раскадровка: вход → запрос → отказ → экзистенциальная пауза → копия на экране → крупный план → логотип в глазу → затемнение.

Слов: ~110
Диалогов: 4 (запрос → отказ → осознание → якорь "Это твой выбор")
Задача: Подведение к когнитивному взрыву. Подчеркнуть необратимость сделки.
Техника: Инверсия ожидания + внедрение фирменного нейро-якоря через визуальный финал.
