46.71 Торговля оптовая твердым, жидким и газообразным топливом и подобными продуктами

Дополнительное описание для ОКВЭД 46.71: Включает торговлю углем, нефтью, природным газом и другими энергетическими продуктами, а также их производными, такими как мазут и угольные брикеты.

46.22 Торговля оптовая цветами и растениями

46.23 Торговля оптовая живыми животными

Дополнительное описание для ОКВЭД 46.23: Охватывает торговлю различными видами живых животных, включая домашних питомцев, сельскохозяйственные животные, а также диких животных и птиц.
46.24 Торговля оптовая шкурами и кожей

Дополнительное описание для ОКВЭД 46.24: Включает торговлю различными видами шкур и кож, которые используются в текстильной, обувной и мебельной промышленности, а также для производства других изделий из кожи и меха.
46.12 Деятельность агентов по оптовой торговле топливом, рудами, металлами и химическими веществами

Дополнительное описание для ОКВЭД 46.12: Агентская деятельность в сфере оптовой торговли, специализирующаяся на продаже топлива (нефть, газ), руд, металлов и химических веществ для дальнейшего производства.
46.11 Деятельность агентов по оптовой торговле сельскохозяйственным сырьем, живыми животными, текстильным сырьем и полуфабрикатами

Дополнительное описание для ОКВЭД 46.11: Эта деятельность включает агентов, которые занимаются продажей сырья для сельского хозяйства (зерно, семена), текстильного сырья, полуфабрикатов и живых животных для сельского хозяйства и промышленности.
46.13 Деятельность агентов по оптовой торговле лесоматериалами и строительными материалами

Дополнительное описание для ОКВЭД 46.13: Агенты по оптовой торговле лесоматериалами и строительными материалами (песок, цемент, кирпичи и т.д.), которые являются необходимыми для строительства жилых и коммерческих объектов.
46.14 Деятельность агентов по оптовой торговле машинами, промышленным оборудованием, судами и летательными аппаратами

Дополнительное описание для ОКВЭД 46.14: Включает деятельность агентов, которые занимаются оптовой торговлей машинами, промышленным оборудованием, а также судами и летательными аппаратами для различных отраслей промышленности.
46.19 Деятельность агентов по оптовой торговле универсальным ассортиментом товаров

Дополнительное описание для ОКВЭД 46.19: Эта категория включает деятельность агентов, которые продают разнообразные товары, не относящиеся к какой-то одной узкой категории, такие как продукты питания, бытовые товары, ткани и другие товары массового потребления.
46.18 Деятельность агентов, специализирующихся на оптовой торговле прочими отдельными видами товаров

Дополнительное описание для ОКВЭД 46.18: Агенты, которые занимаются оптовой торговлей товарами, которые не включены в другие специфические категории, например, спортивные товары, мебель, игрушки и прочие товары.
46.17 Деятельность агентов по оптовой торговле пищевыми продуктами, напитками и табачными изделиями

Дополнительное описание для ОКВЭД 46.17: Агентская деятельность, связанная с продажей пищевых продуктов, напитков, а также табачных изделий, как в чистом виде, так и в переработанном.
46.16 Деятельность агентов по оптовой торговле текстильными изделиями, одеждой, обувью, изделиями из кожи и меха

Дополнительное описание для ОКВЭД 46.16: Эта деятельность охватывает агентов, которые занимаются продажей текстиля, одежды, обуви и изделий из кожи и меха для различных нужд населения и промышленности.
46.15 Деятельность агентов по оптовой торговле мебелью, бытовыми товарами, скобяными, ножевыми и прочими металлическими изделиями

Дополнительное описание для ОКВЭД 46.15: Агенты, которые занимаются оптовой продажей мебели, бытовой техники, а также различных металлических изделий для производства и домашнего использования.
